# pitag (alias package)

This is a name-protection alias for **[pi-tag](https://www.npmjs.com/package/pi-tag)** — the Slack gateway for the [pi coding agent](https://github.com/badlogic/pi-mono).

Installing this package gives you the exact same CLI (it depends on and forwards to `pi-tag`), but prefer the real package:

```bash
npm install -g pi-tag
```

Docs, issues, and source: https://github.com/Crokily/pi-tag
