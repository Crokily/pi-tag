# Security Policy

## Supported Versions

Security updates are provided for the latest released version of pi-tag.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

To report a vulnerability, email colin@aibuild.com with:

- A description of the issue and its potential impact
- Steps to reproduce or proof-of-concept details, if available
- Affected versions or commit hashes
- Any suggested mitigations

We will acknowledge receipt as soon as possible, investigate privately, and coordinate disclosure and remediation with the reporter. Please allow reasonable time for a fix before public disclosure.
